# Rocketry Testing

This repository contains test code for various components used in the rocketry project.

## Projects

### BMP390 Mosfet Test

This project tests the functionality of the BMP390 sensor with a Mosfet.

- `Transmitter.ino`: Code for the transmitter circuit.
- `Reciever.ino`: Code for the receiver circuit.

### BNO085 Mosfet Test

This project tests the functionality of the BNO085 sensor with a Mosfet.

- `Transmitter.ino`: Code for the transmitter circuit.
- `Reciever.ino`: Code for the receiver circuit.
